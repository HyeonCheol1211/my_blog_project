# 🗑️ [Project] 쓰레기통: Spring Boot 학대 시뮬레이터

> **"코딩은 지능 순이라는데, 저는 순위권 밖인 것 같습니다."**

이 프로젝트는 현철이가 **Spring Boot**라는 거창한 이름을 빌려 작성한 **자바 기반의 스파게티 요리법**입니다. 객체지향 원칙(SOLID)을 정면으로 반박하며, 유지보수가 불가능한 코드를 짜는 데 주력했습니다.

---

## 🛠️ 기술적 결함 (Technical Defects)

* **Language:** Java (버전 모름, 그냥 깔려있는 거 씀)
* **Framework:** Spring Boot (사실상 Annotation 발사대)
* **Database:** 없음 (내 머릿속 기억장치보다 휘발성이 강함)
* **Logic:** 복붙(Copy & Paste) 기반의 운빨 코딩

---

## 📂 프로젝트 아키텍처 (Spaghetti Structure)

구조를 이해하려 하지 마세요. 저도 짜면서 길을 잃었습니다.

- `Controller`: 모든 로직이 여기 다 들어있음 (뚱뚱함)
- `Service`: 인터페이스 만들 줄 몰라서 그냥 클래스 하나로 퉁침
- `DTO`: 그게 뭐죠? 그냥 `Map<String, Object>`로 다 보냅니다.
- `Exception`: `try-catch`로 다 감싸고 아무것도 안 함 (침묵의 살인마)
